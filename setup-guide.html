네이버 맵 API 설정 안내

현재 사이트에서 네이버 맵 API 인증 오류가 발생했습니다. 아래 안내에 따라 네이버 클라우드 플랫폼에서 설정을 변경해주세요.

오류 원인

네이버 지도 API는 보안상의 이유로 등록된 웹 서비스 URL에서만 API를 사용할 수 있도록 제한하고 있습니다. 현재 GitHub Pages에서 호스팅 중인 도메인(amu-create.github.io)이 네이버 클라우드 플랫폼에 등록되어 있지 않아 인증 오류가 발생했습니다.

해결 방법

  1. 네이버 클라우드 플랫폼에 로그인합니다.
  2. 콘솔 메뉴로 이동합니다.
  3. AI·Application > Maps 서비스로 이동합니다.
  4. Application 탭에서 현재 사용 중인 애플리케이션을 선택합니다.
  5. Web 서비스 URL 섹션에서 다음 URL을 추가합니다:
    • https://amu-create.github.io
    • https://amu-create.github.io/navermap/
  6. 변경사항을 저장합니다.
  7. 몇 분 후 페이지를 새로고침하여 다시 확인합니다.

로컬 테스트 방법

네이버 클라우드 플랫폼 설정을 변경하기 전에 로컬에서 테스트하려면:

  1. 네이버 클라우드 플랫폼에서 localhost 또는 127.0.0.1을 Web 서비스 URL에 추가합니다.
  2. 간단한 로컬 웹 서버를 실행하여 테스트합니다. 다음과 같은 방법이 있습니다:
    • Python 사용: python -m http.server 명령어 실행
    • Node.js 사용: npx serve 명령어 실행
    • Visual Studio Code Live Server 확장 프로그램 사용
  3. 브라우저에서 http://localhost:8000 또는 서버가 제공하는 URL로 접속합니다.

추가 정보

네이버 지도 API 관련 자세한 내용은 네이버 클라우드 플랫폼 Maps API 문서를 참조하세요.
