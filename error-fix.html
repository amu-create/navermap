네이버 맵 API 인증 오류 해결

오류 상황

네이버 맵 API에서 다음 오류가 발생했습니다:

NAVER Maps JavaScript API v3 네이버 지도 Open API 인증이 실패하였습니다. 클라이언트 아이디와 웹 서비스 URL을 확인해 주세요. * Error Code / Error Message: 200 / Authentication Failed * Client ID: me6liq8hnh * URI: [현재 접속한 URL]

해결 방법

이 오류는 네이버 클라우드 플랫폼에 현재 도메인이 등록되지 않았기 때문입니다. 다음 단계를 따라 해결하세요:

등록할 도메인:
loading...
  1. 네이버 클라우드 플랫폼에 로그인합니다.
  2. 콘솔 > AI・Application > Maps 서비스로 이동합니다.
  3. Application 탭에서 현재 사용 중인 애플리케이션을 선택합니다.
  4. Web 서비스 URL 섹션에서 위에 표시된 도메인을 추가합니다.
  5. 변경사항을 저장합니다.
  6. 약 5-15분 후에 페이지를 새로고침합니다.

주의: 도메인을 등록할 때 마지막 슬래시(/)를 제외하고 등록하세요. 예: https://amu-create.github.io (O), https://amu-create.github.io/ (X)

로컬 테스트하기

로컬 환경에서 테스트하려면 다음 단계를 따르세요:

  1. 네이버 클라우드 플랫폼에 http://localhost:8000 URL을 추가합니다.
  2. 터미널에서 프로젝트 폴더로 이동합니다: cd C:\Users\User\navermaptest
  3. 다음 명령어로 로컬 서버를 실행합니다: python -m http.server
  4. 브라우저에서 http://localhost:8000에 접속합니다.
